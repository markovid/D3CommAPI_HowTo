Tying API Calls Together

  • We now have all the tools we need to create our own character cards and player biographies!.

  • By Using Career Profile calls, we can access a player's information, including their heroes' id numbers.

  • We can make Hero Profile Calls and access all of the heroes' information with their id numbers, including their items' id numbers.

  • Finally, with the items' id numbers, we have access to all of the items' information
