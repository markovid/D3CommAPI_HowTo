The D3 Community API

The D3 Community API will allow us to access information about any diablo 3 player, each of their characters, all of their equipment, and information about in game merchants.

Before we go more indepth about how to use the API calls, lets first set up a mashery account with Blizzard entertainment and get an API key in order to access the information.

First go to dev.battle.net and click register in the upper right hand corner

Proceed to enter you information and create a password, a confirmation email will arrive shortly.

After you follow the link in your confirmation email, you can now apply for access to the API! Just follow the link in step 1.

After you follow the link, fill out the information. Be sure to select the Diablo III Franchise for your api key. Some fields can be left blank without a problem, and the basic plan is sufficient.

We now have access to an API key, Let's see what we can do with it.
